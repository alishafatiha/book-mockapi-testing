# 📚 Book API Testing Project

## 📌 Description
API testing project using Postman for Book Management System.

## 🎯 Scope
- GET all books
- GET book by ID
- POST create book
- PUT update book
- DELETE book

## 🧪 Test Types
- Positive testing
- Negative testing
- Data validation
- Schema validation

## 🛠 Tools
- Postman
- MockAPI

## 📊 Findings
- API accepts invalid data (missing fields)
- price returned as string instead of number
- update endpoint does not persist data

- Postman Collection
- Test Case Document (Google Sheets / Excel)
